# OjAssistant
SustechJcoder平台助手

### 项目结构
```
ojAssistant/
├── main.py                 # 主入口点
├── services/               # 服务层
│   ├── __init__.py
│   ├── auth_service.py     # 认证相关服务
│   ├── data_service.py     # 数据获取服务
│   └── requester.py        # API通信服务
├── ui/
│   ├── __init__.py
│   ├── display.py          # 显示功能
│   └── interaction.py      # 用户交互功能
├── utils/
│   ├── __init__.py
│   ├── formatters.py       # 格式化相关函数
│   └── file_handlers.py    # 文件操作函数
└── config.py               # 配置信息
```

**请合理地正确使用脚本，用于不正当用途[（如暴力刷答案）](https://github.com/JCoder-Pro/FeedBack/issues/6)后果自负**
